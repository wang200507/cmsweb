李旸，读《平凡的世界》

2017-11-19 宁夏水务投资集团有限公司

每个人的生活同样也是一个世界。即使最平凡的人，也得要为他那个世界的存在而战斗。从这个意义上说，在这些平凡的世界里，也没有一天是平静的。”这是《平凡的世界》里面的一句话,这本书令我获益匪浅。

《平凡的世界》是一部现实主义小说，在这篇小说中，作者以他对自己生活的那个年代的了解，以一个农民的视角，高度浓缩了中国西北农村的历史变迁过程，以小见大的使作品达到了思想性与艺术性的高度统一，特别是主人公孙少安的淳朴、无私、面对困境艰苦奋斗带领全村人致富的的精神，让我认识到在那样的年代，在那一片民风淳朴的黄土地上，有这样一群勇敢奋斗、不屈于命运、敢于追求梦想的人们，有着这样一股蓬勃向上、勇于抗争、愈挫愈勇的力量。 在许多年后飞速发展的今天，我们平凡依旧，无论何时，平凡都是世界的基调，也是我们快乐生活的源泉。有追求，我们就向着目标执着、勤奋的努力，而不是只说空话、停滞不前或茫然的无所事事。在这个繁华且躁动的时代，绝大多数人早已能满足温饱，并开始追求更高层次的东西，拥有的东西越来越多，在意的却越来越少了，很多人过着光鲜亮丽的生活，但其灵魂却缺乏安息之地。所以，我们更应该向那个年代的人学习致敬，学习他们淳朴善良的品质；学习他们默默无闻吃苦耐劳的精神；学习他们面对挫折，乐观勇敢的态度；学习他们舍己为人甘于奉献的勇敢……因为，这些优秀的中华传统美德才是我们灵魂的寄托，才是我们的精神财富，才是我们的民族血脉。 一个懂得接受并享受平凡的人，不一定获得多么大的成功，却一定很快乐；不一定完成什么伟大的目标，却一定满怀希望；不一定活的万事如意，却一定拥有一颗坦然面对、荣辱不惊的心。在这个平凡的世界，平凡的你我也许没有靓丽的外表，没有超乎寻常的智慧，也没有纵情山水的情怀，但这并不代表所有的风采会随之远去，因为，平凡的人也能在坚守与执着中度过不平凡的一生。 （长城水务公司 李旸）
