雷锋精神永传承 青年讴歌谱华章

2015-12-25 15:06 宁夏水务投资集团有限公司

——水投集团举办“传承雷锋精神 奏响青春集结号”主题演讲比赛





为纪念雷锋同志逝世51周年，充分学习和发挥雷锋的优良品质，传承、践行雷锋精神，以雷锋精神激发广大员工无私奉献的热情，3月28日，水投集团举办“传承雷锋精神 奏响青春集结号”主题演讲比赛。 13名参赛选手题材各异，角度新颖，个个精神焕发，神采飞扬，用自己的心声讲述着雷锋同志以及身边“雷锋”们一个个动人的故事,发自肺腑的表达对雷锋精神的崇敬，声情并茂的演讲深深感染、启迪着现场的每一位观众。 雷锋精神“是一面永不褪色的旗帜，是一座永放光芒的灯塔，是指引我们前行的宝贵精神财富”,通过“学雷锋”系列活动，既弘扬了雷锋精神，又展现了当代水利人艰苦奋斗，自强不息，助人为乐，服务人民的风采,同时也进一步激发了广大员工爱岗敬业、无私奉献的“螺丝钉”精神，共同为推动集团公司发展贡献力量。>
