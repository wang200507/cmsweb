宁夏精细化工基地一期供水管网工程顺利开工

2017-11-15 宁夏水务投资集团有限公司

11月11日，中宁水务有限公司举办水质检测业务培训班，邀请实验室认证方面专家针对对饮用水和污水检测相关工作开展《检验检测机构资质认定评审准则》培训，公司部分专业技术人员和水质检测中心实验员学习。

授课老师结合水质监测中心具体情况，详细讲授了技术认证知识及流程，现场查看实验室布局、仪器配置，传授业务知识，与水质化验员面对面交流和解答具体问题，为公司取得资质认证奠定基础。

中宁公司2014年6月成立水质检测中心，投资近450万元，购置了原子吸收光谱、气相色谱仪、离子色谱仪等40多套先进仪器。水质监测能力满足《生活饮用水卫生标准》（GB 5749-2006）常规42项指标，能够开展非常规项目和《地下水质量标准》（GB/T 14848-93）检测。取得行业资质认证，不仅满足公司原水、出厂水、管网水、中宁县农村饮用水、工业用水的水质检测需求，还能够对外承接相应水质检测工作。
