水投集团党委关于设立善行义举四德榜的通知

2014-08-12 15:46

集团公司各部门、分公司：

  为深入贯彻落实党的十八大和十八届三中全会精神，扎实推进社会主人核心价值观建设，全面加强四德建设，巩固提升精神文明创建成果，根据水利厅党委的安排部署，集团公司决定设立善行义举四德榜。现将具体实施方案通知如下：

一、指导思想

以 邓小平理论、“三个代表”重要思想、科学发展观为指导，以深入学习贯彻党的十八大、十八届三中全会精神为主线，以社会主义核心价值观体系为根本，遵循思想 道德建设规律，坚持党的群众路线，发挥群众主体作用，员工自我教育与组织培育引导相结合，通过推进善行义举四德榜建设，积极培育和践行社会主人核心价值 观，大力弘扬水利行业精神，引导和激励广大员工以先进模范人物为榜样，营造厚德笃学、贬恶扬善、自尊自律、理性向上的良好风尚，为加快水利现代化建设提供 道德支撑和精神力量。

二、建榜内容

善行义举四德榜围绕家庭美德、职业道德、社会公德、个人品德四个方面，突出社会核心价值观内涵要求，对员工善行义举张榜展示。善行义举榜设置在集团公司《水投》报和水投二级网站。按照孝、诚、爱、仁的分类标准，就四德某项内容建榜。将先进典型的照片和事迹张贴在四德榜上。
（一）家庭美德。以孝老爱亲为切入点，主要内容包括敬老爱老、长幼有序，夫妻和睦、相互忠诚，邻里团结、守望相助等情况。上榜公示以“孝”为核心，如孝敬父母给多少钱、回家帮助父母做什么、在单位表现怎么样等。
（二）职业道德以诚信建设为切入点吗，主要内容宝库奥爱岗敬业、诚实守信、清正廉洁、办事公道等职业操守。上榜公示以“诚”为核心，忠诚事业、忘我工作、追求卓越，诚实劳动、诚信待人。
（三）社会公德。以培育爱心、维护公序良俗为切入点，主要内容包括文明礼貌、助人为乐、见义勇为、爱护公物、保护环境等行为。上榜公示以“爱”我核心，关爱他人、互助友爱，爱护环境、卫生整洁，热心捐赠、无偿献血等。
（四）个人品德。以 涵养正气、履行责任为切入点，主要内容包括爱国励志、好学上进、团结友善、修身律己等。上榜公示以“仁”为核心，知荣辱、分是非、变美丑，感恩父母、感恩 组织、感恩社会、感恩国家、感恩自然，以积极向上的心态对待他人、工作、社会，自觉履行在家庭、岗位和社会应尽的责任和义务。

三、工作流程
（一）知榜：宣传动员阶段。集团公司通过召开动员会等形式，让广大员工明白社会主义核心价值观对个人层面的具体要求即“爱国、敬业、诚信、友善”。通过《水投》报和水投二级网站说明立榜的要求、目的和意义。
（二）议榜：确定标准阶段。集团公司将广泛征求各党支部、工团组织、各部门、分公司意见，讨论确定上榜标准及张榜内容设置。
（三）评榜：规范程序阶段。各部门、分公司按照上榜内容于九月底前提出上榜候选典型报集团综合部。集团公司组织评议，形成初步意见，经党委会审核后公示，评议结果确定后上榜，并上报水利厅党委。
（四）上榜：制作公示阶段。集团公司党委对审定的“善行义举榜”上榜典型张榜展示。并适当给与一定奖励。
（五）用榜：注重实效阶段。集团公司将把评议“善行义举榜”纳入员工教育培训计划，用典型教育引导员工。
（六）管榜：形成常态阶段。要通过逐级筛选、收集各类典型，建立起先进典型储备库，完善先进典型报批、宣传推广、回访帮扶等制度。要通过广泛组织道德主题实践活动，发现培养更多典型，及时维护、更新、完善相关内容，建立起动态管理制度。

四、工作要求
（一）提高认识，落实责任。建立善行义举榜是一项系统工程，知榜、议榜、上榜、用榜、管榜，每一个环节都需要大量细致的工作。建榜不易、坚持更难、重在制度、贵在坚持。集团公司综合部负责具体工作，纳入精神文明建设考核中。
（二）广泛宣传，典型推进。集团各党支部、工团组织每年选树1-2个典型，在《水投》报和水投二级网站开设“四德”榜对先进典型进行宣传，利用微博、微信、QQ群等方式，宣传活动开展情况以及涌现出的好经验、好典型，展示员工队伍的良好风貌。
